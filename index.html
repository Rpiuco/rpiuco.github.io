Trabalhos

piRNAdb

Um banco de dados que amazena sequências e informações relevantes para o estudos dos PIWI-interacting RNAs, uma molécula encontrada em diversos organismos, incluindo humanos. Desenvolvimento completo do sistema, desde coleta, atualização e análise de dados até a execução da codificação. Sistema desenvolvido utilizando PHP, HTML5, CSS, Javascript, muito otimizado para HTML semântico, SEO e performance. Características pouco encontradas em bancos de dados biológicos atuais.

Bioinformatics Lab Logo and Website

Projeto em desenvolvimento. Novo layout, logo e sistema do grupo de Bioinformática pertencente ao Molecular Oncology Center do Hospital Sírio Libanês. Sistema desenvolvido utilizando PHP, HTML5, CSS, Javascript, otimizado para HTML semântico, SEO e performance. Basicamente descreve informações sobre o nosso grupo de pesquisa, bem como áreas de pesquisa, publicações e oportunidades em aberto.

Banco de Dados EMPAG

Sistema desenvolvido como projeto paralelo durante a graduação. Sistema sem acesso externo, para armazenamento de cadastro e informações relativas a alunos estudados pela Dra Angelica Francesca Maris. Desenvolvimento feito em PHP XHTML, CSS, javascript, além de aperfeiçoamento para performance e HTML semântico. Adicionalmente, foi desenvolvido uma ferramenta que possibilitava o desenho de heredrogramas.
